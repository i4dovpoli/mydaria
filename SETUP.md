# Налаштування синхронізації фото для всіх користувачів

Щоб фото, які ви додаєте, були видимі всім користувачам сайту, потрібно налаштувати GitHub Gist API.

## Крок 1: Створення GitHub Personal Access Token

1. Перейдіть на https://github.com/settings/tokens
2. Натисніть "Generate new token" → "Generate new token (classic)"
3. Введіть назву токену (наприклад, "Gallery Token")
4. Виберіть права доступу: **gist** (створення та редагування gist)
5. Натисніть "Generate token"
6. **ВАЖЛИВО**: Скопіюйте токен одразу, він більше не буде показаний!

## Крок 2: Додавання токену до проекту

### Варіант А: Для локальної розробки та GitHub Pages

Відкрийте файл `config.js` та вставте ваш токен:

```javascript
var GITHUB_CONFIG = {
    GITHUB_TOKEN: 'ваш_токен_тут', // Вставте токен сюди
    GIST_ID: (typeof localStorage !== 'undefined' ? localStorage.getItem('galleryGistId') : null) || '',
    GIST_FILENAME: 'gallery-images.json'
};
```

⚠️ **УВАГА**: 
- НЕ публікуйте токен у публічному GitHub репозиторії!
- Якщо ваш репозиторій публічний, використовуйте GitHub Secrets або зберіть токен окремо
- Для приватного репозиторію можна зберігати токен безпосередньо в `config.js`

### Варіант Б: Без токену (тільки localStorage)

Якщо не налаштуєте токен, фото будуть зберігатися тільки локально в браузері кожного користувача. Для синхронізації між користувачами потрібен токен.

## Крок 3: Тестування

1. Відкрийте сайт
2. Натисніть "Додати фото"
3. Введіть пароль: `2799`
4. Завантажте фото
5. Перевірте, що фото з'явилися в галереї
6. Відкрийте сайт в іншому браузері або інкогніто - фото повинні бути там теж!

## Як це працює?

- При першому завантаженні фото створюється приватний GitHub Gist
- Всі завантажені фото зберігаються в цьому Gist
- При відкритті сайту фото завантажуються з Gist
- Базові фото (які в папці `images/`) завжди відображаються
- Додаткові фото синхронізуються між всіма користувачами

## Безпека

- Токен має права тільки на Gist, не на весь акаунт
- Gist створюється як приватний
- Пароль для додавання фото: `2799` (можна змінити в `script.js`)

## Альтернативи

Якщо не хочете використовувати GitHub:
- Firebase Realtime Database (потрібна реєстрація)
- JSONBin.io (простий, але має обмеження)
- Власний backend сервер
